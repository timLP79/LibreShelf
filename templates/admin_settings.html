Patron management

When enabled, staff users gain access to the CSV patron import tool. Other admin-only features (backup / restore, staff management, settings) remain restricted to admins.

External API access

When enabled, LibreShelf will not contact Open Library (or future external metadata sources) for cover art or book information. Book entry continues to work using whatever information staff type in manually. Use this for deployments without internet access or where outbound HTTP is policy-restricted.
