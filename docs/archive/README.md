# Archive

University. Preserved for context and reference. Not maintained -- content here may reference
checkpoints, deadlines, or class assignments that no longer apply.

Contents:

- `canvas-discussion-post.md` -- Hello World assignment submission for the Canvas LMS.
- `tech-stack-survey.md` -- the 03.02 tech stack survey assignment comparing Go, Rails, and ASP.NET.
- `testing-and-debugging-guide.md` -- Go testing tutorial written during the testing-infrastructure milestone.
- `bootstrap-integration-guide.md` -- Bootstrap 5 tutorial written in week 3.
- `go-learning-guide.md` -- personal Go syntax notes accumulated across CP1 through CP7.
- `screenshots/` -- Hello World debugging screenshots from week 3.

Current project documentation lives in [`../`](../) (architecture, security, deployment) and the
root [`README.md`](../../README.md).
